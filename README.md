Логистикс

После клонирования репозитория выполнить команду
```sh
$ npm i
```
Для сборки проекта запустить команду
```sh
$ gulp
```
Запуск в браузере с использованием локального веб-сервера (gulp-webserver)
```sh
http://localhost:8080
```

---
# Используемые модули

 - gulp
 - gulp-autoprefixer
 - gulp-pug
 - gulp-sass
 - gulp-webserver
